# LanguageWire Integrations

Connecting your content ecosystem directly to the LanguageWire platform saves time, reduces manual errors, and speeds up your time-to-market. 

## CMS Connectors

We offer out-of-the-box connectors for major Content Management Systems. The most popular include:
- **Sitecore**: Seamlessly send pages and components for translation directly from the Sitecore interface.
- **Optimizely (Episerver)**: Full support for Optimizely Commerce and CMS.
- **Adobe Experience Manager (AEM)**: Native integration handling complex XML structures and fragments.

## PIM Integrations

Product Information Management (PIM) systems are crucial for e-commerce.
- **inriver PIM**: Translate product descriptions, attributes, and specifications without leaving the inriver platform.
- **Akeneo**: Automated workflows to handle massive catalogs of products.

## Custom API

If you use a custom system, our REST API allows you to build your own integration. You can automate project creation, upload files, check status, and download deliverables programmatically using OAuth 2.0 authentication.
