# Getting Started with Acme Translate

Acme Translate is a cloud platform for managing translation projects. This guide covers your first project.

## Creating a project

1. From the dashboard, click **New Project**.
2. Choose a source language and one or more target languages.
3. Upload your files. Each file may be up to **50 MB**; larger files should be split before upload.
4. Select a workflow: **Machine Translation**, **MT + Human Review**, or **Human Translation**.

## Tracking progress

The project page shows per-file status: *Queued*, *In Translation*, *In Review*, and *Completed*. You'll receive an email notification when all files reach *Completed*.

## Downloading results

Completed translations can be downloaded individually or as a single ZIP archive from the **Deliverables** tab. Translated files keep the original formatting and file name, with the target language code appended (e.g., `manual_de-DE.docx`).

## Getting help

Use the in-app chat for support, or consult the other help articles for details on file formats, terminology management, and user roles.
